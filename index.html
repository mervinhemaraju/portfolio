The Mauritius Emergency Services App uses a RESTful API that provides a list of all emergency contacts in Mauritius, along with cyclone information such as precautionary guides and current cyclone levels. This API was built from scratch and is hosted on my personal cloud.

LockMe

Lock Me is an unofficial android application that allows you to lock your phone without using the physical button on your device. It makes use of the new Accessiblity Services API provided by Android to send announcement events when a user press the lock button provided by the app.
